# Stutter.js

RFC-5389 Stun/Turn Node.js implementation

## Example
```js
    var stutter = require('./').Stun; 
    var Stun = new stutter({port:1234, 
                            debug:true, 
                            authentication:'none'}); 
    Stun.createServer(); 
```
